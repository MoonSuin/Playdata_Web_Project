    - 성공한 것 
        - 재료별 레시피 검색 
        - 질병별 레시피 검색       
       

    - 실패한 것 
        - 관리자 모드로 검색 키워드들 변경, 추가
        - 재료와 질병 두가지 키워드에서 하나씩 선택하면 다 적용해서 조회 
        - 여러 재료 선택해서 정보 조회
            -> 만들지 못한 이유 : 
                >> 검색시 키워드가 하드 코딩 되어있는 것도 있지만 관리자 모드로 값을 추가해줬을 때 코드에 적용 시킬 수 있는 방법 구상하기가 어려웠다
                >> 시간 들여서 처리해볼 생각이다 
       
            -> 개선사항 : 검색 키워드 DB를 만들어서 정보를 불러오는 방식으로 처리해보면 될 것 같다.
       
        - 이미지 상대 좌표로 저장 
            -> 방법 찾는 시간의 부족 
            -> 개선 사항 찾으면서 방법을 찾아 해결했다.
